This is a Video sharing social media app(tik tok clone). 

This app let user's to log on to there account.
It is a full stack project where people can upload, publish, share and comment on videos. Also people can search by categories.
Used technologies: ReactJS, Typescript, NEXTjs, OAuth, Sanity.

You can find this deployed website in:- https://shotnshareit.vercel.app

Here people can logIn using there Goodle Account.So that they can upload videos. Also it pulls the google account photo as profile photo.
